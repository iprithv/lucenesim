# Contributing

Pull requests welcome. Open an issue first if you're planning a big change.

## Setting up

```bash
./gradlew test
```

That's it. No external dependencies to install - the vendored Lucene jars are in `libs/`.

## Before submitting

- `./gradlew test` should pass
- Keep the code style consistent with what's there
- Add a test if you're fixing a bug or adding a feature
